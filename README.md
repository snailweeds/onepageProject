# 프로젝트 서버에 올리기 - 배포
* IP주소: 컴퓨터마다 가지는 고유 주소, 네트워크가 가능한 모든 기기가 통신할 수 있도록 가지는 특수한 번호
* 포트: 하나의 IP에는 여러 포트 존재, 하나의 포트에 하나의 프로그램 실행 가능
* URL: IP주소를 알파벳으로 바꾼 것
* DNS: IP주소를 URL로 변환해주는 시스템
